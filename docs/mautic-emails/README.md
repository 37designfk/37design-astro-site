## Mautic投入手順

1. Mautic管理画面 or APIでメールテンプレートを作成
2. フォームを3つ作成（AI診断/ROI/サイト診断）
3. キャンペーンを3つ作成（フォーム送信トリガー → 5通のステップメール）
4. ツールページの formId を実際のIDに更新

## 配信スケジュール（全シリーズ共通）

- 1通目: 即時（フォーム送信直後）
- 2通目: 2日後
- 3通目: 5日後
- 4通目: 8日後
- 5通目: 12日後
