こんにちは、37Design代表の古田です。**週77時間 Claude Code を業務運用** に使い、14プロダクトを動かしている現役AI社長です (直近1週間ピーク値、次の目標は月300時間到達)。

「Claude (クロード) を触ってはみたけど、結局 ChatGPT と何が違うのか分からない」「業務で使いこなしている人は何をしているのか」——非エンジニア経営者の方からよく頂く質問です。

この記事では、**経営者・非エンジニアが本当に使う機能だけ**に絞って、Claudeの使い方を体系的に解説します。Claudeの全機能カタログではなく、**毎日業務で動かしている運用者の手の動き**を再現するイメージで読んでください。

> Claudeの概要・モデル構成は[Claude(クロード)とは?中小企業経営者のためのClaude AI完全ガイド](/blog/claude-ai-complete-guide-sme-2026/)で解説しています。

## ステップ1 - アカウント作成とプラン選択

### claude.ai にアクセスして登録

ブラウザで [claude.ai](https://claude.ai/) を開き、Google アカウント or メールアドレスでサインアップします。これで**無料のFreeプラン**が使えるようになります。

ただし業務利用するなら **Free は捨てて Pro ($20/月) から始める**のが正解です。Free は1日のメッセージ数が極端に少なく、ファイルアップロードもできないため、業務利用にならないからです。

### 個人 vs 法人プラン

- **個人で試す段階**: Pro ($20/月) を経費で1人契約
- **業務に組み込んで複数人で使う**: Team ($25-30/月 × 人数)
- **ヘビーユーザー1人で集中運用**: Max ($100-200/月) ← 私はここ
- **API連携で自社システムに組み込み**: API従量課金

詳しい料金構造は[Claude料金プラン徹底比較【Pro/Max/Team/Enterprise/API】](/blog/claude-pricing-plans-complete-comparison-2026/)を参照してください。

## ステップ2 - 基本のチャット入力 (経営者の最初の3パターン)

Claude を初めて触る経営者がまず試すべき3つの使い方です。

### パターンA - 文章のリライト・整形

メール・提案書・SNS投稿の下書きを貼り付けて、「丁寧な敬語で書き直して」「もっと簡潔に」「中小企業経営者向けのトーンで」と指示するだけ。**5秒で人が書いた文章の品質が1段上がります**。

### パターンB - 議事録の構造化

打ち合わせのメモを丸ごと貼り付け、「決定事項・宿題・期日・関係者の4軸で整理して」と指示。手作業30分が3秒に。

### パターンC - 競合・市場情報の整理

「中小企業向けAI顧問サービスの主要プレイヤーを5社、特徴と価格帯を表で整理して」のような調査依頼。完璧ではないですが**80%の精度のたたき台**が一瞬で出てきます。

## ステップ3 - ファイル添付で「自社情報を知っているClaude」を作る

Claude が他のAIと一線を画すのは **長文ファイル添付の処理能力 (200,000トークン = 約15万字)** です。

### 添付できるファイル

- PDF (契約書・マニュアル・パンフレット)
- Word/Excel/PowerPoint
- テキスト/CSV/Markdown
- 画像 (スクリーンショット・図表・名刺)

### 経営者が最初にやるべき「自社情報の食わせ込み」

以下の自社資料を1つのチャットにアップロードして、「これがうちの会社です」と教えてください:

1. 会社案内 PDF
2. 料金表
3. サービス内容のWebページをコピペしたテキスト
4. 過去の顧客対応FAQ
5. 自社の事業計画書 (機密性に配慮)

これで Claude は **「あなたの会社のことを理解した社員」**として動くようになります。以降のチャットで「うちの会社で○○の問い合わせが来たらどう返す?」と聞くと、自社のトーン・料金体系を踏まえた回答が返ってきます。

### 機密情報の取り扱い注意

Pro 版は会話データが学習に使われる可能性があるため、**顧客の個人情報・特許前の機密情報は入れない**こと。学習に使われないのは **API利用** または **Enterprise プラン**です。

## ステップ4 - Projects 機能で「業務別Claude」を量産する

Pro 以上のプランで使える **Projects (プロジェクト)** 機能は、業務別に「専用のClaude」を作れます。

### Projects の作り方

1. claude.ai 左メニューの「Projects」 → 「+ Create project」
2. プロジェクト名 (例: 「営業メール返信」「ブログ記事生成」「議事録構造化」)
3. プロジェクトに**Knowledge** (自社資料をアップロード) と **Instructions** (この Project の役割を文章で記述) を設定

### 私の Projects 例

- **議事録自動整形** - Instructions に「決定事項/宿題/期日/関係者の4軸で構造化、固有名詞は伏字にしない」
- **クライアント月次レポート** - Knowledge に GA4/GSC レポートテンプレ、Instructions に「経営者向け1ページ要約 + 改善提案3つ」
- **営業メール返信** - Knowledge に料金表 + 過去の対応事例、Instructions に「丁寧だがビジネスライクな返答、断る時は理由を簡潔に」

これだけで、毎日の業務スピードが3〜5倍になります。

## ステップ5 - Artifacts で「使える成果物」を出す

Claude には **Artifacts** という、画面右側に成果物 (コード/HTML/Markdown/SVG) を独立表示する機能があります。

### Artifacts で何ができるか

- HTML/CSS で簡易LPを試作 (画面に即プレビュー)
- 業務マニュアルをMarkdownで作って即DL
- 顧客向けの比較表をテーブルで生成
- 簡易SVG図表 (組織図、フロー図) を生成

非エンジニアの経営者でも、**「LP のドラフトを作って」「業務フロー図を作って」と言うだけで形になる**のが Artifacts の威力です。

## ステップ6 - Skills と MCP - 上級者向けの拡張機能

ここからは **業務本気組み込み段階**の機能です。

### Skills (スキル) - 繰り返し業務のテンプレ化

Claude Code (CLIツール) に組み込まれている機能で、自分専用の「スキル」(処理パッケージ) を作れます。例えば私の場合「Claude Code の使用時間を集計するスキル」「営業録音から議事録を作るスキル」「WordPressに記事を投稿するスキル」など、数百のスキルを動かしています。

### MCP (Model Context Protocol) - 外部サービス連携

MCP は **Claude と外部サービス (Gmail / Google Calendar / Stripe / GitHub / WordPress 等) を繋ぐ標準プロトコル**です。MCPサーバーを設定すると、Claude が直接「Gmail を読み」「Stripe で請求書を作り」「WordPress に記事を投稿」できるようになります。

私自身、47以上のMCPサーバーを動かして、業務の大半をClaude経由で実行しています。

詳しいClaude Codeの世界は[Claude Codeとは?経営者・非エンジニア向け完全ガイド](/blog/claude-code-complete-guide-for-business-owners-2026/)を参照してください。

## 経営者がClaudeを業務に組み込む「最初の30日間」プラン

順序を決めて段階的に習熟するのが最短です。

| Day | やること |
|---|---|
| 1-3 | Pro 登録、文章リライト・議事録整形を1日10回ずつ試す |
| 4-7 | 自社資料をアップロードして「自社を知っているClaude」を作る |
| 8-14 | Projects を3つ作る (営業メール / 月次レポ / マニュアル整形) |
| 15-21 | Artifacts で簡易LP・比較表・フロー図を3つ試作 |
| 22-30 | Claude Code (CLI) のインストールを試す (オプション) |

30日後、ほとんどの経営者は **「もうClaudeなしでは仕事にならない」**状態になります。

## Claude の使い方でよくある質問 (FAQ)

### ChatGPTと使い分けはどうすればいいですか?

長文処理・コード生成・社内ドキュメント整理は **Claude**、画像生成・ブラウジング検索は **ChatGPT** が定石です。私は8割の業務をClaudeで処理し、画像生成だけChatGPTを使っています。

### 日本語の精度はどうですか?

2025年以降、Claudeの日本語精度はChatGPTと並ぶ業務利用品質です。原稿執筆・契約書ドラフトまで日本語で問題なく運用できます。

### 機密情報を入れても大丈夫ですか?

Pro/Max は学習データに使われる可能性があるため、機密情報は **API利用または Enterprise プラン** を選択してください。

### Claude を社内で複数人で使うには?

Team プラン ($25-30/月 × 人数) を契約し、管理者がメンバーを招待します。Projects はチーム内で共有可能です。

### Claude の使い方を体系的に学ぶ方法は?

公式ドキュメントは英語が中心です。日本語で体系的に学びたい場合は、現役運用者の解説記事 + 実際に毎日触ることの組み合わせが最短です。37Designでは経営者向けのClaude活用講座を準備中です。

## まとめ - Claudeは「業務に乗る」段階で本領を発揮する

Claude は「便利なチャットAI」として触っているうちは、ChatGPT と差を感じにくいかもしれません。しかし、**Projects / Artifacts / ファイル添付 / Claude Code / MCP** まで踏み込むと、業務処理スピードが桁違いに変わります。

経営者が今やるべき3アクション:

1. **Pro ($20/月) を契約して、今日1時間使い倒す**
2. **自社資料を5本アップロードして「自社を知っているClaude」を作る**
3. **業務別 Projects を3つ作って、定型業務を任せる**

これだけで、来月の業務効率が大きく変わります。
